Estimarr el valor de: \[I = \int_{0}^{1} x^2 \sqrt{1-x^2} \,dx = \dfrac{\pi}{16} \simeq 0,1963495408493621\]

Mtodo Montecarlo

Para estimar esta integral mediante el Mtodo de Montecarlo, hay que poner la integral como una esperanza de x.

Es decir: \(I=E[g(X)]\), donde X es una variable uniforme en el intervalo (0,1) y g(x) = \(x^2 \sqrt{1-x^2}\).

  1. Establecemos la forma de generar valores aleatorios, en este caso de manera uniforme en el intervalo \((0, 1)\).

  2. Definimos la funcin \(g\) que deber aplicarse a esos valores aleatorios.

genera_valor_aleatorio <- function() {
  runif(1)
}



g <- function(x) {
  sqrt((1 - x^2))*x^2
}
  1. Replicamos el proceso de generar un valor aleatorio y aplicarle la funcin \(g\). La variable n determina la cantidad de replicaciones.
n <- 1e5
valores_g <- replicate(n , {
  x <- genera_valor_aleatorio()
  g(x)
}
)
  1. Calculamos una estimacin de \(I_{1}\) a partir de la media aritmtica de los valores obtenidos.
estimacion <- mean(valores_g)

El valor estimado de \(I\) es \(0.1968812\). Este mtodo siempre dar una aproximacin, mejor o peor pero se trata de una aproximacin.

Puesto que esto es un resultado aleatorio, hay que calcular siempre un intervalo de confianza que permita acotar la variabilidad de la estimacin.

probabilidad_cobertura <- 0.997
alfa <- 1 - probabilidad_cobertura
percentil<- qnorm(1 - alfa / 2)
error_estandar <- sqrt(var(valores_g)/n)
intervalo_confianza <- estimacion + c(-1,1) * error_estandar * percentil

Luego, un intervalo de confianza con probabilidad de cobertura 0.997 es ( \(0.195603, 0.1981593\) )

Muestreo estratificado: asignacin proporcional

  1. En primer lugar definimos los estratos y establecemos la forma de generar valores aleatorios dentro de cada estrato. Una manera simple de definir los estratos es subdividir el intervalo (0, 1) de forma independiente en cada dimensin.
estratos <- data.frame(
  min = c(0, 4 / 5),
  max = c(4/ 5, 1),
  probabilidad = c(4 / 5, 1 / 5)
)
# Funcin que genera valores dentro de un estrato
genera_valor_en_estrato <- function(numero_estrato) {
  estrato <- estratos[numero_estrato, ]
  runif(1, min = estrato$min, max = estrato$max)
}
  1. Ahora replicamos el proceso de generar valores en cada estrato, en una cantidad proporcional a su probabilidad, y aplicarles la funcin g a cada uno de ellos. Haremos tambin uso del paquete bench para estimar el coste en tiempo del mtodo, para poder as estimar su eficiencia a la hora de estimar el valor de la integral.
unidad_de_tiempo <- "ms"
n_estratos <- n * estratos$probabilidad
# Aseguramos valores enteros
n_estratos <- ceiling(n_estratos)
# Aseguramos al menos dos valores en cada estrato
n_estratos <- pmax(n_estratos, 2)

coste_estratificado_proporcional <- bench::mark(
  {
    valores <- lapply(
      seq_len(2), # (1,2), aplico la funcin 2 veces, una para cada estrato
      function(numero_estrato) {
        replicate(n_estratos[numero_estrato], {
          x <- genera_valor_en_estrato(numero_estrato)
          g(x)
        })
      }
    )
  },
  iterations = 10,
  time_unit = unidad_de_tiempo
)$median

Se han generado \(8000\) valores en el primer estrato y \(2000\) valores en el segundo estrato.

  1. Finalmente, estimamos el valor de la integral, la varianza de esa estimacin y la eficiencia del mtodo.
estimacion_estratificado_proporcional <-
  weighted.mean(
    sapply(valores, mean),
    estratos$probabilidad
  )
varianza_estratificado_proporcional <-

eficiencia_estratificado_proporcional <-
  1 / (varianza_estratificado_proporcional *
    coste_estratificado_proporcional)

Muestreo estratificado: asignacin ptima

Consideramos los mismos estratos que antes y, por tanto, la misma forma de generar valores en cada uno de ellos.

Ahora replicamos el proceso de generar en cada estrato una cantidad ptima de valores, determinada mediante un procedimiento en dos etapas, y aplicarles la funcin g a cada uno de ellos. Haremos tambin uso del paquete bench para estimar el coste en tiempo del mtodo, para poder as estimar su eficiencia a la hora de estimar el valor de la integral.

n_tanteo <- 100
n_produccion <- n - n_tanteo # Para que la cantidad total de valores generados sea igual en los tres mtodos y, por tanto, su comparacin tenga sentido

coste_estratificado_optimo <- bench::mark(
  {
    # Estimacin de las varianzas de los estratos
    n_estratos <- pmax(
      ceiling(n_tanteo * estratos$probabilidad), # asignacin proporcional para calcular la var
      2
    )
    valores <- lapply(
      seq_len(2),
      function(numero_estrato) {
        replicate(n_estratos[numero_estrato], {
          x <- genera_valor_en_estrato(numero_estrato)
          g(x)
        })
      }
    )

    # Cantidad ptima de valores en cada estrato
    sigmas <- sapply(valores, sd)
    n_estratos <-
      pmax(
        ceiling(n_produccion * estratos$probabilidad * sigmas /
          sum(estratos$probabilidad * sigmas)),
        2
      )

    # Generacin de valores en cada estrato
    valores <- lapply(
      seq_len(2),
      function(numero_estrato) {
        replicate(n_estratos[numero_estrato], {
          x <- genera_valor_en_estrato(numero_estrato)
          g(x)
        })
      }
    )
  },
  iterations = 10,
  time_unit = unidad_de_tiempo
)$median

Reduccin de la varianza por Muestreo por Importancia:

A continuacin vamos a aplicar el mtodo del muestreo por importancia para tratar de reducir la varianza de la estimacin.

Buscamos una densidad instrumental con soporte al menos en el intervalo \((0, 1)\) y que sea lo ms parecida posible a la funcin \(|g(x)| f_{1}(x)\), donde \(f_{1}\) es la densidad de \(\mathrm{U}(0, 1)\). Puesto que la funcin de densidad de una distribucin beta de parmetros \(\alpha\) y \(\beta\) es \(f_2(x) \propto x^{\alpha - 1} (1 - x)^{\beta - 1}\), con soporte en \((0, 1)\), parece conveniente usar como densidad instrumental la de una distribucin beta con \(\alpha = 3\) y \(\beta = \frac{1}{2} + 1 = \frac{3}{2}\).

library(ggplot2)

alfa <- 3
beta <- 3 / 2
ggplot(data.frame()) +
  geom_function(fun = g, aes(colour = "g")) +
  geom_function(fun = dunif, aes(colour = "unif")) +
  geom_function(
    fun = dbeta,
    args = list(shape1 = alfa, shape2 = beta),

En primer lugar, establecemos la forma de generar valores aleatorios, que ahora ser a partir de la distribucin beta escogida. Por otra parte, el producto de \(g\) por la razn de verosimilitud (es decir, el cociente entre la funcin de densidad de la distribucin uniforme y la funcin de densidad de la distribucin beta escogida) es el siguiente:

Recuerda, la funcin de densidad de la distribucin beta es la siguiente:

\[X \sim \beta(\alpha,\beta), \quad f_X(x)= \dfrac{x^{\alpha-1}(1-x)^{\beta-1}}{B(\alpha,\beta)}\]

Adems, \[\quad B(\alpha,\beta)= \dfrac{\Gamma(\alpha)\Gamma(\beta)}{\Gamma(\alpha + \beta)} \]

Por tanto:

\[ g(x) \frac{1}{\frac{x^{3 - 1} (1 - x)^{3/2 - 1}}{B(3, 3/2)}} = \frac{\Gamma(3) \Gamma(3/2)}{\Gamma(5/2)} {x^{3 - 1} (1 - x)^{3/2 - 1}} \]

genera_valor_aleatorio <- function() {
  rbeta(1, shape1 = alfa, shape2 = beta)
}

g_por_verosimilitud <- function(x) {
  (gamma(3)*gamma(3/2)/gamma(5/2))*(x^2)*(sqrt(1-x))
}
